## Logo Files

Place your RT NEWS logo here.

**Recommended Files:**
- `rt-news-logo.png` - Main logo (PNG format recommended)
- `rt-news-logo.svg` - Vector logo (SVG format for scalability)
- `rt-news-logo-dark.png` - Dark theme version
- `rt-news-logo-light.png` - Light theme version

**Usage in Code:**
```jsx
<img src="/images/logo/rt-news-logo.png" alt="RT NEWS Logo" />
```

**File Size Tips:**
- Keep under 500KB for PNG
- SVG is best for logos (scales without quality loss)
- Recommended dimensions: 200x100px or maintain aspect ratio
