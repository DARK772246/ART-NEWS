## CEO Profile Picture

Place your CEO profile picture here.

**Recommended Files:**
- `ceo-profile.jpg` - Main CEO photo
- `ceo-profile.png` - High-quality PNG version
- `ceo-profile-thumb.jpg` - Thumbnail version (200x200px)

**Usage in Code:**
```jsx
// In AboutUs page or CEO section
<img src="/images/ceo/ceo-profile.jpg" alt="CEO" className="rounded-full" />

// With different sizes
<img src="/images/ceo/ceo-profile-thumb.jpg" alt="CEO Thumbnail" />
```

**Photo Requirements:**
- Square format recommended for profile pictures
- Minimum 300x300px resolution
- Professional headshot preferred
- JPEG or PNG format
- Keep file size under 1MB for web optimization

**Display Ideas:**
- Profile section on About Us page
- About section in Footer
- Team members section (if adding more team)
- CEO quote or bio section
