## Logo & Images Directory

This directory is for storing your logo and other images for the website.

### Image Placement Guide:

1. **Logo Image**
   - Place your logo file here (e.g., `logo.png`, `logo.svg`)
   - Reference in code: `/images/logo.png`

2. **Header Images**
   - Place any header-related images here
   - Reference in code: `/images/header-image.png`

3. **Article Images**
   - Default article thumbnails can go here
   - Reference in code: `/images/default-article.jpg`

4. **Social Media Icons**
   - If needed, place custom social icons here
   - Reference in code: `/images/social/facebook.png`

5. **CEO/Team Photos**
   - CEO and team member photos
   - Reference in code: `/images/team/ceo.jpg`

### How to Use in Code:

```jsx
<img src="/images/logo.png" alt="Logo" />
<img src="/images/team/ceo.jpg" alt="CEO Photo" />
```

### Note:
- All images will be served from the `/images/` path
- Recommended image formats: PNG, JPG, SVG
- Optimize images for web to improve loading speed
- Keep file sizes reasonable (compress when possible)
